TAGS

(Voltar)

As tags são ferramentas usadas para se criar algo em html, sendo assim, de extrema importância que seja vista e revista em todas as ocasiões. Tendo como principais as tags:
< html > é o elemento raiz que define todo o documento HTML
< head > contém metadados sobre o título e a codificação de caracteres da página.
< body > inclui todo o conteúdo que aparece na página.

TAGS DE TEXTO

(Voltar)

As tags de textos são aquelas criadas para inserir textos em seu html, tendo como principais:
< H1,H2,H3,H4... > São aquelas para adicionar títulos, subtítulos e etc.
< P > É aquela que serve para adicionar novos paragráfos.
< BLOCKQUOTE > Criada para adicionar sitações.

ADIÇAO DE LINKS

(Voltar)

Para adicionar um link em seu html, deverá usar a seguinte tag:
< a heref="ENDEREÇO DO LINK" >... < / A >
